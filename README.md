# Luyện Tập Thuật Toán Ứng Dụng - Đại học Thủy lợi

Repository tổng hợp các bài toán và bài thực hành môn **Thuật Toán Ứng Dụng** tại Trường Đại học Thủy lợi.

## Nội dung chính

Project tập trung vào các kỹ thuật giải thuật quan trọng thường gặp trong chương trình học và các bài tập thực hành:

- Quy hoạch động (Dynamic Programming)
- Chia để trị (Divide and Conquer)
- Quay lui (Backtracking)
- Tối ưu thuật toán
- Phân tích độ phức tạp

---

# Các chuyên đề

## 1. Quy hoạch động (Dynamic Programming)

Bao gồm các bài toán:
- Dãy con tăng dài nhất (LIS)
- Ba lô 0/1 (Knapsack)
- Fibonacci tối ưu
- Đường đi ngắn nhất
- Xâu con chung dài nhất (LCS)

Kỹ năng luyện tập:
- Xây dựng công thức truy hồi
- Tối ưu bộ nhớ
- DP 1 chiều / 2 chiều

---

## 2. Chia để trị (Divide and Conquer)

Bao gồm:
- Merge Sort
- Quick Sort
- Binary Search
- Tìm phần tử lớn nhất / nhỏ nhất
- Chia mảng và xử lý đệ quy

Kỹ năng:
- Tư duy đệ quy
- Phân tách bài toán
- Tối ưu thời gian xử lý

---

## 3. Quay lui (Backtracking)

Bao gồm:
- Sinh hoán vị
- Sinh tổ hợp
- N-Queens
- Sudoku Solver
- Maze / Mê cung

Kỹ năng:
- Cây trạng thái
- Cắt tỉa nhánh
- DFS và tìm kiếm lời giải
